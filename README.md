# React Three Fiber Hero Planet Component

A performant, interactive 3D hero section featuring a floating planet with subtle mouse parallax tilt, bloom glow, and responsive design. Built with **React Three Fiber**, **@react-three/drei**, and **@react-three/postprocessing**.

Perfect for modern portfolio websites, agency landing pages, or any project needing an eye-catching animated background.

## Features

- Smooth mouse-interactive parallax tilt (feels like grabbing the planet)
- Gentle floating animation
- Optimized bloom post-processing for subtle glow
- Responsive scaling (smaller on mobile)
- High performance: `frameloop="demand"`, capped DPR, preset environment
- Clean, modern lighting with warm golden tones
